Excercise 4. a. Dr. Healy advises against using pie charts, he described them as perceptually bad and explained that the human visual system is better at decoding lengths from a fixed baseline than decoding angles.

  1. Wilke argues that piecharts are better at showing how parts combine to form a majority or whole which other forms of visualization are limited in doing.

  2. Wilke presented side-by-side bars and stacked bars as alternatives to pie charts. He prefers side-by-side bars when the goal is to directly compare the individual fractions to each other. On the other hand, he favors stacked bars for visualizing many sets of proportions or time series of proportions, as they clearly show the proportions of a whole while allowing for comparisons across different conditions.

my_spec <- list(
  `$schema` = "https://vega.github.io/schema/vega-lite/v5.json",
  data = list(values = list(
    list(party = "CDU/CSU", seats = 243),
    list(party = "SPD",     seats = 214),
    list(party = "FDP",     seats = 39)
  )),
  layer = list(
    list(
      mark = list(type = "arc", outerRadius = 150, stroke = "white", strokeWidth = 3),
      encoding = list(
        theta = list(field = "seats", type = "quantitative", stack = TRUE),
        color = list(
          field = "party",
          type = "nominal",
          scale = list(
            domain = list("CDU/CSU", "SPD", "FDP"),
            range  = list("#555555", "#a63030", "#e8d44d")
          ),
          legend = NULL
        )
      )
    ),
    list(
      mark = list(type = "text", radius = 100, fontSize = 16, fontWeight = "bold", color = "white"),
      encoding = list(
        theta = list(field = "seats", type = "quantitative", stack = TRUE),
        text = list(field = "seats", type = "quantitative")
      )
    ),
    list(
      mark = list(type = "text", radius = 175, fontSize = 13),
      encoding = list(
        theta = list(field = "seats", type = "quantitative", stack = TRUE),
        text = list(field = "party", type = "nominal")
      )
    )
  )
)
